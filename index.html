About Programming

What is Programming?

This is the process of designing and building an executable computer program to accomplish a specific computing result or to perform a specific task. Programming involves stuffs such as:analysis, generating algorithms, profiling algorithms, accuracy and resource consumption, and the implementation of algorithms in a chosen language.

The source code of a program is written in one or more languages that are intelligible to programmers, rather than machine code, which is directly executable by th CPU.

The perpose of programming is to find a sequence of instructions that will automate the performance of a task, often for solving problems.
